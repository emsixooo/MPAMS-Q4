# Multi Purpose Attendance Monitoring System
## *Computer Science - 2 (Final Project)*

# Description
- An automated attendance monitoring system built with Python, utilizing NoSQL databases via `.json` files.
- An attendance tracker that generates reports and provides a detailed summary of attendance patterns.

# Objectives
- Automate attendance tracking through Python.
- Provide daily, weekly, and monthly attendance reports for students and employees.
- Develop a reliable and user-friendly program to make managing attendance easier and more convenient.

# Functions
- **Create, Read, Update, and Delete** attendance entries (CRUD functionality).
- Generate weekly and monthly summaries of absences and attendance percentages.
- Export data from `.json` format to a spreadsheet (MS Excel `.xlsx` file).

# Prerequisites
- **Python 3.x**.
- **Pandas library** (for data manipulation and Excel support).
- **openpyxl library** (required by Pandas for exporting `.xlsx` files).
