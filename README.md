# Project Title
# multi purpose attendance monitoring system *Computer Science - 2 (Final Project)*
# Description
- An automated attendance monitoring system with Python, using databases (NoSQL) (use of .json)
- A attendance tracker, that generates reports, and provide a summary of attendance patterns.
# Objectives
- Automate attendance through Python.
- Provide daily, weekly, and monthly attendance reports for students / employees.
- Make a reliable and user friendly program to make managing attendance more easier and convenient.
# Functions
- Create, Read, Update, Delete attendance entries.
- Create weekly, monthly summaries of absences.
- Export data in .json to spreadsheet (MS Excel .xlsx file)
